# Technology Stack: FoodPal

## Core Technologies
- **Programming Language:** Dart (optimized for modern, performant mobile apps).
- **Frontend Framework:** Flutter (providing a consistent experience across all platforms).
- **State Management:** Riverpod (`flutter_riverpod`) for robust and scalable state handling.

## Essential Libraries
- **Google Fonts:** For high-quality typography.
- **Cupertino Icons:** For clean, iOS-style iconography.
- **Gemini API:** For advanced, AI-powered food recognition and nutritional analysis.

## Development Standards
- **Standardized UI Widgets:** Reusable custom widgets to maintain consistency.
- **Clean Architecture:** Clear separation of data models, providers (business logic), and UI screens.
- **Linting:** Standard Flutter lints for high-quality, maintainable code.

## Platforms
- **Primary:** Mobile (iOS/Android)
- **Secondary:** Web and Desktop support (responsive design)
